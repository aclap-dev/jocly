Fantastic XIII.

Qui a dit qu’une variante du jeu d’échecs devait se baser sur un tablier avec un nombre pair de lignes et de colonnes? Pourquoi ne pas partir sur le nombre impair : 13? C’est l’idée de départ qui pilote cette variante. Un jeu impair se devant d'être différent. C’est dans sa nature. Donc, l'armée au combat dans ce jeu présente des troupes qui ont été peu utilisés dans les autres variantes, bien qu'il soit facile de s’y familiariser pour les nouveaux joueurs. Comme on pouvait s’y attendre, il y a 13 types de pieces dans ce jeu. Certaines d’entre elles ont un rayon d’action très long (jusqu’à 3 cases) ce qui peut être pertubant. Cela donne une atmosphere étrange. C‘est tout bonnement fantastique.
